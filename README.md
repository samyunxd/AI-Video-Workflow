## 🚀 Quick Start

1.  **Identity**: Paste your **Gemini API Key** in the header. (Stored locally for your session).
2.  **Script**: Input your narrative in the **01. Script** box.
3.  **Aesthetic**: Define the visual "anchor" in **02. Style** (e.g., *70s anamorphic, hazy dawn*).
4.  **Execute**: Click **Generate Sequences** to begin the synthesis.

## 🛠 Features Mastery

### 📦 Global Context (01-04)
- **Script**: 100% literal text coverage. No skips, no paraphrasing.
- **Style**: The "Art Direction" master. This is context-bonded to every scene for consistency.
- **Negative**: Red-colored exclusions (e.g., `text, watermark, low quality`).
- **Instructions**: Specific logic overrides (e.g., `Focus on silhouette lighting`).

### 🎨 Prompt Modes (05)
*Choose how descriptions are linguistically structured:*

| Mode | Format | Best For... |
| :--- | :--- | :--- |
| **Structured** | `Shot, Subject, Atmosphere, Lighting` | Tag-based models (default). |
| **General** | `Narrative descriptive paragraph.` | LLM-ready/Natural language models. |
| **Graphic** | `Vector art, flat design, geometric.` | Illustrations and brand assets. |
| **JSON** | `{"shot": "wide", "mood": "dark"}` | Post-processing / Data pipelines. |

### ⚙️ Engine Optimization (06)
*Content is tuned to match specific training data biases:*

- **`Flux`**: Prioritizes complex physics and photorealistic surface textures.
- **`Midjourney`**: Leans into "Aesthetic" modifiers and high-concept moods.
- **`ST. Diffusion`**: Uses prioritized token weights and community keywords.
- **`Gemini`**: Conversational depth and high semantic accuracy.

### ⏱ Dynamics & Motion (07-09)
- **Temporal Control**: Set `Seconds per Scene` and `Words/Sec` to define pacing.
- **Strict Mode**: `[Zap Icon]` Eliminates metaphorical fluff for high-precision engines.
- **Camera/Motion**: Multi-select lens types and motion paths to randomize cinematic variety.

## 💡 Pro-Tips for Cinematic Results

### 1. The "Stability" Anchor
Descriptions change per scene, but your **Style** remains constant. Use it for "Film Stock" (e.g., *Kodak Vision3 500T*) to keep colors consistent across a 5-minute video.

### 2. Pacing the Narrative
If scenes feel rushed, lower the **Words per Second** (e.g., to `1.5`) or increase **Duration**. This creates more rows with less text, giving each image "breathing room."

### 3. Workflow Integration
Generated prompts and negative constraints are **merged on copy**. When you click copy on an image prompt, it automatically appends the negative prompt in a format your engine understands.
